# Contributing (開発への協力について)

このプロジェクト（DevLog）に興味を持っていただきありがとうございます！
もし改善の提案やバグの報告がある場合は、以下の手順でご協力いただけます。

## 協力の方法

### 1. 提案や報告をする（Issues）
何か気づいたことや、追加してほしい機能があれば、GitHub上の「Issues」から新規報告を作成してください。
*   どのような問題があるか、またはどのような機能が欲しいかを分かりやすくお書きください。

### 2. 直接ファイルを修正する（Pull Requests）
もしご自身でファイルを修正して提案したい場合は、以下の手順で行うことができます。
1. このプロジェクトをご自身のアカウントにコピー（Fork）します。
2. 変更用の新しい作業用ブランチ（Branch）を作成します。
3. ファイルを修正し、保存（Commit）します。
4. 修正内容を提案（Pull Request）として送信してください。

---

ご協力ありがとうございます！
