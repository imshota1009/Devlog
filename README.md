## 概要
「使いたいと思うものをつくる」をモットーに、Shotaが独学で制作したスマートフォンアプリ（iOS / Android）を展示・紹介しているポートフォリオサイトです。
作品ごとの独自の世界観やデザインにこだわり、丁寧に磨き上げたアプリを掲載しています。

## 掲載されている主な作品
*   **TaviNote (旅行管理アプリ / Android & iOS)**
    *   旅行の準備タスクをこなすたびに「旅の木」が成長していく、新しい形の旅行しおり・管理アプリです。
    *   旅の計画、日記、持ち物リスト、思い出の記録まで、旅行に必要なすべての機能を一冊のしおりのようにまとめられます。
*   **Next App (近日公開)**
    *   現在、次の新しいスマートフォンアプリケーションを企画・開発中です。

## サイトの目的
このポートフォリオサイトは、制作したスマートフォンアプリの魅力的な紹介、ダウンロードリンクへの案内、そして制作者（Shota）自身の自己紹介や連絡先（SNS）をまとめた総合的な発信拠点として作られています。
